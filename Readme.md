# Microwave Timer (VHDL)

## Overview
This project implements a digital microwave timer using VHDL. The system allows users to set a time in minutes and seconds and simulates the behavior of a real microwave oven.

## Features
- Countdown timer (MM:SS)
- User input via switches (BCD)
- Start mechanism via push button
- Door control (pause/resume)
- End-of-count signal

## Architecture
The system is divided into:
- Datapath: counters for minutes and seconds
- Control Unit: FSM

## Technologies
- VHDL
- Quartus II
- FPGA (DE1)
